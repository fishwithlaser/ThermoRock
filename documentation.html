ThermoCalc

python script for calculating possible thermodynamics reactions of minerals.
This script was largely developed based on the doctoral research developed
by Dr. Xu Chu (xu.chu@utoronto.ca) and adapted to be able to scale to large
quantities of possible reactions. Mostly coded using procedural programming.

Structure

There are a few features of the code with a few packages. Briefly they are:

main folder

requirments.py

documents required packages. Can be run from bash or powershell. Allows
easy integration of virtual environments :).

thermorock.py

run this to complete the script regularly
